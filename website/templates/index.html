Placeholdy

Easily generate placeholder images

Placeholdy is a simple tool/website for generating placeholder images. It helps you to embed placeholder images with no hassles.

Psst... it is customizable: see below!

Minimal working example

When embedding an image, link to {{ config.public_url }}/placeholder?width=512&height=512...

..or in HTML: <img> alt="..." src="{{ config.public_url }}/placeholder?width=512&height=512"</img>

The numbers are in pixels format.

Customizing colors

By adding the URL parameters &text_color and/or &background_color, you can customize the color of the placeholder image.

Supported color formats

Placeholdy uses Pillow to generate images under the hood. See ImageColor's color names.

Hint: Hex, RGB, and HTML color names are all supported.

Note: Remember to escape hex colors in the URL: &text_color=%23398727 instead of &text_color=#398727.

More information
